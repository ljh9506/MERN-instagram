# React-MERN SHOP


```
React와 Express.js로 풀스택 개발한 Instagram Clone 사이트입니다.
팔로우 팔로워 기능, 이미지게시물 업로드 및 삭제, 좋아요 및 댓글, 회원가입
메일을 통한 비밀번호찾기, 무한스크롤 기능을 적용하였습니다.
```

### 사용한 플러그인
```
- React.js
- Context-API
- Material-UI
- Axios
- Express.js
- MongoDB
- Cloudinary 
- bcrypt
- multer
- nodemailer
```
